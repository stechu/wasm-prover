zkSNARK Prover in WASM
=======================

## Dependencies:
* [Rust toolchain](https://www.rust-lang.org/tools/install)
* [npm](https://www.npmjs.com/get-npm)
* `wasm-pack` package:
    ```bash
    cargo install wasm-pack
    ```

## Run the benchmark
```bash
./serve.sh
```
You can view the result at `localhost:8080`.
